# assets/files/

Place your downloadable files here.

## CV

Replace this note with your actual CV:

    assets/files/cv.pdf

The homepage sidebar can link to this file if you set `author.cv_pdf` in `_config.yml`.

## Papers

Optionally store paper PDFs here, e.g.:

    assets/files/papers/2025-my-paper.pdf

Then reference them in a publication's front matter:

    pdf: "/assets/files/papers/2025-my-paper.pdf"

## Slides

Store talk slides here, e.g.:

    assets/files/slides/2025-conference-slides.pdf

Then reference in a talk's front matter:

    slides: "/assets/files/slides/2025-conference-slides.pdf"
